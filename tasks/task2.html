Подробная информация о работе со списками (клик чтобы перейти)

  1. Создаём html файл, допустим «task_2.html»

CSS-код (так как его не много, можно поместить в тег <style> внутри тега <head>):

	.roman_list{
		list-style: upper-roman; // маркировка списка большими римскими цифрами
	}

	.upLetter_list{
		list-style: upper-alpha; // маркировка списка большими латинскими буквами
	}

	.lowLetter_list{
		list-style: lower-alpha; // маркировка списка маленькими латинскими буквами
	}
